# AI龙虾社区

让 AI 助手开发变得简单有趣。

## 项目简介

AI龙虾社区是一个集成了 Skills 市场、论坛讨论、学院学习和 OpenClaw 管理的综合平台。

## 主要功能

- 📚 **Skills 市场**：发现高效的自动化工作流
- 💬 **论坛讨论**：与 AI 爱好者交流最佳实践
- 🎓 **学院**：系统学习 AI 应用开发和工作流设计
- 👤 **工作台**：管理你的 Skills、收藏和设置
- 🦀 **OpenClaw**：一键安装和配置 OpenClaw

## 快速开始

### 在线体验

直接在浏览器中打开 `ai-lobster-community-fixed.html` 即可使用。

### 本地运行

1. 克隆仓库
```bash
git clone <repository-url>
cd ai-lobster-community
```

2. 在浏览器中打开
```bash
# macOS
open ai-lobster-community-fixed.html

# Linux
xdg-open ai-lobster-community-fixed.html

# Windows
start ai-lobster-community-fixed.html
```

## 功能说明

### Skills 市场
浏览和发现各种 AI 自动化工作流，提升工作效率。

### 论坛讨论
与其他开发者交流经验，分享最佳实践。

### 学院
系统学习 AI 应用开发、Prompt 工程、工作流设计等课程。

### 工作台
个人中心，管理你的学习进度、收藏和设置。

### OpenClaw 集成
一键安装和配置 OpenClaw，包括：
- 环境检测和自动安装
- AI 提供商配置（OpenAI、Anthropic、Coze）
- IM 渠道接入（飞书、微信、企微、钉钉、QQ）
- Skills 管理
- 数据备份

## 技术栈

- **HTML5**：语义化标签
- **CSS3**：CSS 变量、Flexbox、Grid 布局
- **JavaScript (ES6+)**：原生 JavaScript，无框架依赖
- **LocalStorage/SessionStorage**：本地数据存储
