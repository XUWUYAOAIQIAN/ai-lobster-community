## 技术细节

### 新增 CSS（~135 行）
- `auth-tabs` - 登录/注册标签样式
- `profile-content` - 个人资料样式
- `profile-avatar` - 头像样式
- `profile-stats` - 统计信息样式
- `avatar-grid` - 头像选择器样式

### 新增 JavaScript（~130 行）
- `handleRegister()` - 处理注册
- `handleLogout()` - 退出登录
- `switchAuthTab()` - 切换标签
- `clearAllErrors()` - 清除错误
- `openProfileModal()` - 打开资料
- `closeProfileModal()` - 关闭资料
- `changeAvatar()` - 更换头像
- `saveProfile()` - 保存资料

### 数据结构
```javascript
{
  id: 'u' + Date.now(),
  username: '用户名',
  email: 'user@example.com',
  avatar: '🦞',
  bio: '个人简介',
  createdAt: '2026-03-30T23:55:00.000Z',
  installedSkills: [],
  skillsUsed: 0
}
```

### 存储
- `localStorage.lobster_users` - 所有用户
- `sessionStorage.lobster_user` - 当前用户
- `localStorage.installed_skills` - 已安装 Skills
