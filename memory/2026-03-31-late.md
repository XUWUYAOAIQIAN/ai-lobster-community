## 技术细节

### 新增 CSS（~100 行）
- `forum-action-btn` - 社交按钮
- `forum-action-btn.liked` - 点赞状态
- `post-detail-content` - 帖子详情
- `comments-section` - 评论区域
- `comment` - 单条评论
- `comment-form` - 评论表单
- `forum-header` - 论坛头部

### 新增 JavaScript（~310 行）
- `toggleLike(postId)` - 点赞
- `showPostDetail(postId)` - 帖子详情
- `closePostDetail()` - 关闭详情
- `addComment(postId)` - 添加评论
- `createNewPost()` - 创建新帖
- `closeCreatePost()` - 关闭创建
- `submitPost()` - 提交新帖
- `saveCurrentUser()` - 保存用户

### 数据结构
```javascript
// 帖子
{
  id: 1,
  title: '帖子标题',
  content: '帖子内容',
  author: '作者',
  avatar: '🦞',
  time: '2小时前',
  likes: 234,
  replies: 45,
  comments: [
    {
      id: 'c1',
      author: '作者',
      avatar: '🐱',
      content: '评论内容',
      time: '1小时前'
    }
  ]
}
```

### 存储
- `localStorage.liked_posts` - 已点赞列表
- `localStorage.forum_posts` - 所有帖子
- `sessionStorage.lobster_user` - 当前用户
