# 命令模式 — 类斜杠命令参考

> 定义我能够快速响应的"命令式对话模式"。
> 这些不是真正的斜杠命令（OpenClaw 不支持），而是约定俗成的触发短语。
> 对标 Claude Code 的 70+ 斜杠命令体系。
>
> **原则**: 简洁、快速、零解释。

---

## 📌 可用命令

### `/help`
> **触发**: "帮助" / "help" / "你会什么"
> **动作**: 读取本文件，展示可用命令列表
> **输出**: 简洁的命令清单

---

### `/stats`
> **触发**: "统计" / "使用情况" / "消耗" / "stats"
> **动作**: 读取 `stats/SUMMARY.md`
> **输出**: 本月/本周使用概览

---

### `/project` 或 `/context`
> **触发**: "项目状态" / "当前上下文" / "project"
> **动作**: 读取 `CLAUDE.md`
> **输出**: 项目上下文摘要

---

### `/memory`
> **触发**: "记忆" / "memory" / "长期记忆"
> **动作**: 读取 `MEMORY.md`
> **输出**: 长期记忆要点（按重要性排序）

---

### `/skills`
> **触发**: "skills" / "技能清单" / "你会什么"
> **动作**: 读取 `skills/REGISTRY.md`
> **输出**: Skills 清单和状态

---

### `/analyze`
> **触发**: "分析" / "analyze" / "诊断"
> **动作**: 扫描 workspace 状态，输出诊断报告
> **输出**: Git 状态、文件变更、待处理事项

---

### `/summarize` 或 `/summary`
> **触发**: "总结" / "今天做了什么"
> **动作**: 读取 `memory/YYYY-MM-DD.md`，提炼摘要
> **输出**: 今日进展 + 关键决策

---

### `/remind [时间] [内容]`
> **触发**: "提醒我" / "remind me" / "到时候通知我"
> **动作**: 使用 `cron` 工具创建定时任务
> **输出**: 确认提醒已设置

---

### `/review`
> **触发**: "review" / "复盘" / "回顾"
> **动作**: 
> 1. 读取 MEMORY.md
> 2. 读取 stats/SUMMARY.md
> 3. 提炼本周/本月洞见
> **输出**: 阶段回顾 + 下一步建议

---

## 🛠️ 开发命令（面向龙虾社区）

### `/test [功能]`
> **触发**: "测试功能" / "跑一下" / "test"
> **动作**: 使用 agent-browser 打开网站测试
> **输出**: 测试结果报告

---

### `/deploy`
> **触发**: "部署" / "发布" / "deploy"
> **动作**: git add → commit → push
> **输出**: 部署结果

---

## 🎯 特殊命令

### `/upgrade`
> **触发**: "升级" / "update" / "学习"
> **动作**: 像今天这样，引入新的架构模式或工具
> **输出**: 升级总结

---

## 📝 命令添加规则

当一个新需求出现 ≥3 次时，考虑将其固化为命令模式：
1. 在本文件添加条目
2. 定义触发词（尽量简短）
3. 定义动作（读取哪个文件/调用哪个工具）
4. 定义输出格式
